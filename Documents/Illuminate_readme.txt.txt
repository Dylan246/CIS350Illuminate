Game Name: Illuminate
By: Dylan Gazda, Samuel Dwyer, Ryan Egan, Brian Hartwig
Course: CIS 350
Contact email address: dgazda@mail.bradley.edu

Basic Controls:
Movement: WASD, or arrow keys
Jumping: A or [space bar]
Equipping lights: E
Put lights on hangers: E
Aim lights: Move mouse

Useful Controls:
Restart level: R
Pause (within a level): ESC
Quit (from main menu): ESC
Quit to main menu from level: Q

How to Win?
Use various light sources to find the exit door in each level.
